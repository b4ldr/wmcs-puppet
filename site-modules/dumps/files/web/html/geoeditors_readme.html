Analytics Datasets: Geoeditors

Contents

This data set measures editor activity on a specific wiki from a specific country. We publish approximate ranges instead of precise numbers and exclude a few countries from the results. You can read about the data and the reasoning behind it on our on-Wiki documentation.

Updates

This data set is updated monthly, usually within one or two days of the first day of each month. It will include data for the month that just concluded, in a separate file. The file name will look like: geoeditors-monthly-YYYY-MM.tsv
