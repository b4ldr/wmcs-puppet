Analytics Datasets: Unique Devices

Statistics compiled using the Unique Devices Definition are available to download here, since 2016.

This data estimates the count of unique devices that visit our sites without assigning unique IDs to users, in order to protect their privacy. These files present, for every wiki project (e.g. Armenian Wikipedia) and every project family (e.g. All Wiktionaries) three figures: the uniques underestimate, the estimate, and the offset. There's more info about these numbers in the last access solution for uniques wiki page. More details about the method can be found in the definition linked above and the blog post announcing it: Unique Devices Dataset.

Download Unique Devices Data

Other ways to access the data are detailed in the definition. For example, we have a Unique Devices API that allows programatic access to smaller amounts of data.
