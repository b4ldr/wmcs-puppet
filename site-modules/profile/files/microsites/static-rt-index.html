This is the static, read-only HTML version of Wikimedia Request Tracker.

Request Tracker used to be a ticket system used by Operations.

You can find old tickets and historical activity in this static HTML version. Example links:

  • Ticket 1

Replace '1' with your bug number.

Links in the original RT format such as ./Ticket/Display.html?id=123also work and are internal redirects to the HTML files.

So you can simply add "static-" to old RT URLs to get here.
