What is this?

This is a collection of legacy dashboards (most of which have been retired) and tools for exploring work and data from the Wikimedia Search Platform and the former Wikimedia Discovery teams. It contains everything from API usage to direct user interactions, and will only grow over time. The idea is that this will provide data for internal and external use, to see how well we're doing.

Who built it and how?

This project was developed by Mikhail Popov and Chelsy Xie (former contributor) on top of RStudio's Shiny web application framework for R. Os Keyes was the original developer.

Who maintains it?

This project is no longer actively maintained. It is owned by the Wikimedia Product Analytics team.
